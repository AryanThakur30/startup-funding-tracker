# Indian Startup Funding Tracker

An AI-powered and data-driven platform to analyze Indian startup funding trends in real time.

## Features
- Funding trend analysis
- Investor tracking
- Interactive dashboard
- Automated data pipeline
- Startup ecosystem insights

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB / SQL
- Tailwind CSS

## Future Goals
- Real-time API integration
- Predictive funding analytics
- VC recommendation engine
