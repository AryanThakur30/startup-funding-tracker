## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/AryanThakur30/startup-funding-tracker.git
cd startup-funding-tracker/india-startup-funding-tracker
```

---

### 2. Install Backend Dependencies

```bash
cd backend
npm install
```

---

### 3. Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

### 4. Configure Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
MONGO_URI=your_database_connection_string
JWT_SECRET=your_secret_key
```

---

### 5. Start Backend Server

```bash
cd backend
npm start
```

Backend server will run on:

```bash
http://localhost:5000
```

---

### 6. Start Frontend

Open a new terminal:

```bash
cd frontend
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```
